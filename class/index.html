Eventos de Mouse

click

Explicação: disparado quando um elemento é clicado.

Sobre o event
Atributo Descrição Uso
target Elemento que disparou o evento event.target.id
type Tipo do evento event.type
timeStamp Tempo em milissegundos desde o carregamento da página event.timeStamp
Outros Outros atributos do evento event.clientX, event.clientY, etc.
dblclick

Explicação: disparado quando dois cliques rápidos são feitos.

Sobre o event
Atributo Descrição Uso
target Elemento que disparou o evento event.target.id
detail Número de cliques consecutivos event.detail
timeStamp Tempo em milissegundos desde o carregamento da página event.timeStamp
mousedown

Explicação: disparado quando um botão do mouse é pressionado.

Sobre o event
Atributo Descrição Uso
button Qual botão do mouse foi pressionado (0: esquerdo, 1: meio, 2: direito) event.button
buttons Estado dos botões do mouse event.buttons
clientX/clientY Coordenadas do mouse relativas à janela event.clientX, event.clientY
mouseup

Explicação: disparado quando um botão do mouse é liberado.

Sobre o event
Atributo Descrição Uso
button Qual botão do mouse foi liberado event.button
target Elemento que disparou o evento event.target
clientX/clientY Coordenadas do mouse relativas à janela event.clientX, event.clientY
mousemove

Explicação: disparado quando o mouse é movido.

Sobre o event
Atributo Descrição Uso
clientX/clientY Coordenadas do mouse relativas à janela event.clientX, event.clientY
movementX/movementY Movimento relativo desde último evento event.movementX, event.movementY
screenX/screenY Coordenadas do mouse relativas à tela event.screenX, event.screenY
mouseover

Explicação: disparado quando o mouse passa por cima de um elemento.

Sobre o event
Atributo Descrição Uso
target Elemento que disparou o evento event.target
relatedTarget Elemento de onde o mouse veio event.relatedTarget
clientX/clientY Coordenadas do mouse relativas à janela event.clientX, event.clientY
mouseout

Explicação: disparado quando o mouse sai de cima de um elemento.

Sobre o event
Atributo Descrição Uso
target Elemento que disparou o evento event.target
relatedTarget Elemento para onde o mouse foi event.relatedTarget
clientX/clientY Coordenadas do mouse relativas à janela event.clientX, event.clientY
mouseenter

Explicação: disparado quando o mouse entra em um elemento.

Sobre o event
Atributo Descrição Uso
target Elemento que disparou o evento event.target
clientX/clientY Coordenadas do mouse relativas à janela event.clientX, event.clientY
timeStamp Tempo em milissegundos desde o carregamento da página event.timeStamp
mouseleave

Explicação: disparado quando o mouse deixa um elemento.

Sobre o event
Atributo Descrição Uso
target Elemento que disparou o evento event.target
clientX/clientY Coordenadas do mouse relativas à janela event.clientX, event.clientY

Eventos de Teclado

keydown

Explicação: disparado quando uma tecla é pressionada.

Sobre o event
Atributo Descrição Uso
key Valor da tecla pressionada event.key
code Código físico da tecla event.code
keyCode Código numérico da tecla (deprecated) event.keyCode
keyup

Explicação: disparado quando uma tecla é liberada.

Sobre o event
Atributo Descrição Uso
key Valor da tecla liberada event.key
code Código físico da tecla event.code
repeat Se a tecla estava sendo repetida event.repeat
keypress

Explicação: disparado quando uma tecla é pressionada e liberada.

Sobre o event
Atributo Descrição Uso
key Caractere gerado pela tecla event.key
charCode Código Unicode do caractere (deprecated) event.charCode
which Código da tecla (deprecated) event.which

Eventos de Formulário

submit

Explicação: disparado quando um formulário é enviado.

Sobre o event
Atributo Descrição Uso
target O formulário que disparou o evento event.target
preventDefault() Método para prevenir envio padrão event.preventDefault()
submitter Elemento que iniciou o envio event.submitter
change

Explicação: disparado quando o valor de um elemento de formulário é alterado.

Sobre o event
Atributo Descrição Uso
target Elemento que teve valor alterado event.target
target.value Novo valor do elemento event.target.value
timeStamp Momento da mudança event.timeStamp
input

Explicação: disparado quando o conteúdo de um elemento de formulário é alterado.

Sobre o event
Atributo Descrição Uso
target Elemento sendo editado event.target
data Caracteres inseridos event.data
inputType Tipo de entrada (insertText, deleteContent, etc) event.inputType
focus

Explicação: disparado quando um elemento recebe o foco.

Sobre o event
Atributo Descrição Uso
target Elemento que recebeu o foco event.target
relatedTarget Elemento que perdeu o foco event.relatedTarget
timeStamp Momento do foco event.timeStamp
blur

Explicação: disparado quando um elemento perde o foco.

Sobre o event
Atributo Descrição Uso
target Elemento que perdeu o foco event.target
relatedTarget Elemento que recebeu o foco event.relatedTarget
timeStamp Momento da perda de foco event.timeStamp
