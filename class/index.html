Eventos de Mouse

click

Explicação: disparado quando um elemento é clicado.

Sobre o event
Atributo Descrição Uso
target Elemento que disparou o evento event.target.id
type Tipo do evento event.type
timeStamp Tempo em milissegundos desde o carregamento da página event.timeStamp
Outros Outros atributos do evento event.clientX, event.clientY, etc.

Eventos de Teclado

keydown

Explicação: disparado quando uma tecla é pressionada.

O objeto event contém event.key, event.code, etc.

Eventos de Formulário

submit

Explicação: disparado quando um formulário é enviado.

O objeto event permite event.preventDefault().

Eventos de Janela

load

Explicação: disparado quando a janela ou recurso é carregado.

Eventos de Touch

touchstart

Explicação: disparado quando um dedo toca a tela.

Outros Eventos

contextmenu

Explicação: disparado ao abrir o menu de contexto.
